# 🏎️ PITWALL — F1 AI Dashboard

An advanced **Formula 1 Analytics & AI Dashboard** built using **Streamlit, Machine Learning, and real-world race data**.

---

## 🚀 Features

* 📊 **Driver Performance Analysis**
* 📈 **Race-by-Race Performance Trends**
* 🤝 **Driver vs Teammate Comparison**
* 🤖 **AI-Based Finishing Position Prediction**
* 🏆 **Win Probability Prediction**
* 🧠 **AI Driver Ranking System**
* 🗺️ **Global F1 Circuit Map**
* 🔐 **Login Authentication System**
* 🎨 **Premium F1-style UI (PITWALL Design)**

---

## 🧠 Machine Learning Models

* **Random Forest Regressor**

  * Predicts finishing position based on grid position

* **Random Forest Classifier**

  * Predicts win probability

---

## 🛠️ Tech Stack

* **Python**
* **Streamlit**
* **Pandas & NumPy**
* **Plotly (Interactive Visualizations)**
* **Scikit-learn (ML Models)**

## ▶️ How to Run

### 1. Install dependencies

```
pip install -r requirements.txt
```

### 2. Run the app

```
python -m streamlit run app.py
```

### 3. Open in browser

```
http://localhost:8501
```
